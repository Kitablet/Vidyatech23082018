------Readme File-------------------------------

Folder Structure
----------------

Cloud Server - Contains the code for Cloud Server and API
eReader - Contains the code of Kitablet App and Browser Version
Database - Contains the backup of a clean database

----------------------------------------------------------------------------

Database connectivity
---------------------
Edit file: Cloud Server\FISE API\FISE API\Web.config
Line Number: 8
Value: Replace SERVERNAME with Database server name, DATABASENAME with Database name, USERID with user id to connect to the db and PASSWORD with password for the user id

----------------------------------------------------------------------------

API PATH
--------

Edit Following files:
- eReader\Apps\FISE\FISE\FISE\Constant.cs
- eReader\Browser\FISE_Browser\FISE_Browser\Web.config
- Cloud Server\FISE Cloud\FISE Cloud\Web.config

Value: replace SERVERNAME with Hosted Server Name for API
